Counter: (Dis)Integration

Steps

  1. Ensure that you every number like type( e.g. int, float, etc ) is behind some typedef.
  2. Instead original type-number( int, float, etc ) use:
    ::counter::number

Example

Original code

void myFunction( float f ){ ... }

typedefed float

typedef float MyFloat;
void myFunction( MyFloat f ){ ... }

Integrate

typedef ::counter::number<float> MyFloat;
void myFunction( MyFloat f ){ }

Disintegration

typedef float MyFloat;
void myFunction( MyFloat f ){ }
