Counter: (Dis)Integration

Steps
  1. Ensure that you every number like type( e.g. int, float, etc ) is behind some typedef.
  2. Make two or more empty struct.
  3. Instead original type-number( int, float, etc ) use:
    ::counter::number< number, empty_struct >
    Where number is: int, float, etc.
    Where empty_struct is some of previously created emptu struct.
Example:
Original code
void myFunction( float f ){ ... }
void myProcess( float f ){ ... }
typedefed floats
typedef float MyFirstFloat;
typedef float MySecondFloat;
void myFunction( MyFirstFloat f ){ ... }
void myProcess( MySecondFloat f ){ ... }
Add empty class
struct MyFirstCategory{};
struct MySecondCategory{};
typedef float MyFirstFloat;
typedef float MySecondFloat;
void myFunction( MyFirstFloat f ){ ... }
void myProcess( MySecondFloat f ){ ... }
Integrate
struct MyFirstCategory{};
struct MySecondCategory{};
typedef ::counter::number<float, MyFirstCategory> MyFirstFloat;
typedef ::counter::number<float, MySecondCategory> MySecondFloat;
void myFunction( MyFirstFloat f ){ ... }
void myProcess( MySecondFloat f ){ ... }
Disintegration
struct MyFirstCategory{};
struct MySecondCategory{};
typedef float MyFirstFloat;
typedef float MySecondFloat;
void myFunction( MyFirstFloat f ){ ... }
void myProcess( MySecondFloat f ){ ... }
